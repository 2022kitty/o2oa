TopMethod: select
打开一个窗口，用于选择视图中的数据。
Syntax:
Arguments:
view : （object）必选，json格式对象，要获取数据的视图信息，数据格式如下：
viewName : （string）视图名称或别名。
application : （string）应用名称或别名。
isTitle : （string）“yes”或者“no”，是否显示视图标题。默认为“yes”
select : （string）“multi”或者“single”，表示多选或单选。默认为“multi”
callback : （function）事件回调方法
option : （object）可选，json格式对象，选择窗口属性，数据格式如下：
width : （number）窗口的宽度。
height : （number）窗口的高度。
title : （string）窗口的标题栏显示内容。
